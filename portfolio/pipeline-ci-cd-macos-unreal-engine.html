UltiMage – Pipeline de Build CI/CD no macOS com Unreal Engine
🧩 Descrição do Projeto

Automatização do processo de build no macOS para o jogo UltiMage, desenvolvido pela Crystal Realms Entertainment. A pipeline CI/CD foi configurada com GitHub Actions e um runner auto-hospedado, garantindo maior estabilidade e redução de erros manuais no desenvolvimento do jogo.

🎯 Objetivo do Projeto

O objetivo foi agilizar o processo de onboarding de desenvolvedores na Crystal Realms Entertainment, automatizando a configuração de chaves SSH no Windows. Isso reduziu etapas manuais, minimizou erros de configuração e acelerou o acesso aos repositórios do GitHub.

🚀 Tecnologias Utilizadas
Unreal Engine 5 GitHub Actions CI/CD macOS RunUAT Self-hosted Runners

Configuração da Pipeline

Configurei uma pipeline de integração contínua utilizando GitHub Actions e um runner auto-hospedado para automatizar o processo de build no macOS do jogo UltiMage, desenvolvido em Unreal Engine 5.

Pipeline

A pipeline executa o comando local RunUAT.sh BuildCookRun, gerencia dependências e referências de módulos (como StructUtils) e arquiva automaticamente os artefatos de build.

Runner Auto-hospedado

O runner auto-hospedado foi configurado como um serviço com o script svc.sh para garantir sua disponibilidade constante.

Resultado

Essa automação estabilizou o processo de build e minimizou a intervenção manual durante o desenvolvimento. Além disso, auxiliou no pinpointing de problemas e na resolução de bugs.
